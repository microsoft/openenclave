# oe_spin_unlock()

Release the lock on a spin lock.

## Syntax

    int oe_spin_unlock(oe_spinlock_t *spinlock)
## Description 

A thread calls this function to release a lock on a spin lock.



## Parameters

#### spinlock

Unlock this spin lock.

## Returns

Return zero if successful
