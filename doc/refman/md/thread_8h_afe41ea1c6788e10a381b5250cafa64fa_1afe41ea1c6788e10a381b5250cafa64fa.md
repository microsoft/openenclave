# oe_spin_destroy()

Destroy a spin lock.

## Syntax

    oe_result_t oe_spin_destroy(oe_spinlock_t *spinlock)
## Description 

This function destroys a spin lock.



## Parameters

#### spinlock

Destroy this spin lock.

## Returns

OE_OK the operation was successful

## Returns

OE_INVALID_PARAMETER one or more parameters is invalid
