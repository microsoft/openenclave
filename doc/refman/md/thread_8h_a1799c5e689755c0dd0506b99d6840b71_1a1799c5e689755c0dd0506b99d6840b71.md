# OE_CondDestroy()

Destroys a condition variable.

## Syntax

    int OE_CondDestroy(OE_Cond *cond)
## Description 

This function destroys the given condition variable.



## Parameters

#### cond

Destroy this condition variable.

## Returns

Returns zero on success.
