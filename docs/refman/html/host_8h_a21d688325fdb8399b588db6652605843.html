oe_result_t oe_parse_report ( const uint8_t *  report,
uint32_t  reportSize,

Parse an enclave report into a standard format for reading.

Parameters
reportThe buffer containing the report to parse.
reportSizeThe size of the report buffer.
parsedReportThe oe_report_t structure to populate with the report properties in a standard format. The parsedReport holds pointers to fields within the supplied report and must not be used beyond the lifetime of the report.
Return values
OE_OKThe report was successfully created.
OE_INVALID_PARAMETERAt least one parameter is invalid.
